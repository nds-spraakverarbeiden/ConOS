Handschriften:
- [x] Handschrift M (Monacensis), München Cgm 25, ca. 850
	(vv. 85-2198a, 2256-2514a, 2576-3414a, 3491-3950, 1017a-4574, 4740b-5275a), by Behaghel considered as the most authentic manuscript, but because it has gaps, the BT text had to normalize the spelling (Price 2010, p.51)
- [x] Handschrift C, London Cotton Caligula A VII, ca. 970
	(vv. 1-5968), probably copied in Southern England, occasional Old English spelling, esp. in C.255-265, contains a large number of superficial errors (Price 2010, p.15)
- [x] Fragment S, Straubing, um 850
	(vv. 351-360, 368-384, 393-400, 492-582, 675-683, 693-706, 715-722)
- [x] Fragment P, Berlin/ehemals Prag R 56/2537, um 850
	(vv. 958-1006), shares with C an error in v.980.
- [x] Fragment V, Vatikan Pal. Lat. 1447, um 880
	(vv. 1279-1358), found together with the Old Saxon Genesis, preserves verse 1308 more correctly than C and M
- [x] Fragment L (Leipzig, Universitätsbibliothek, St. Thomas 4073 (Ms.))
	Price (2010) speculates that P and L may actually represent the same original Ms.
	(vv. 5823-5870)

Stemma (simplified after Price 2010, p.20)

*A (archetype)
	-> V
	-> *PL/C/M
		-> *PL/C
			-> C
			-> *L
				-> P
				-> L
		-> *M/S
			-> M
			-> S 

On the basis of a historically attested exchange of Saxon and Anglo-Saxon literature betweem Mathilde II of Essen and Æthelweard, Freise (quoted by Price, 2010, p.215) argues that transition of C to England occured around 980. Price (2010, p.215) also mentions the 929 engagement of Edith (sister of King Æthelstan) to Otto the Great as an earlier possibility, but either way the later would be the terminus ante quem for *PL/C
